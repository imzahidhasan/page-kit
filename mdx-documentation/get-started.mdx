# Get Started

Welcome to Page Kit. This guide helps you get productive fast.

## What is Page Kit?

A lightweight collection of accessible UI components and docs scaffolding.

## Features

- Simple component registry
- MDX based documentation
- Dark mode ready
- Keyboard accessible navigation

## Quick Start

1. Install dependencies
2. Start the dev server
3. Add or edit MDX docs under `components/mdx-documentation`

```bash
npm install
npm run dev
```

Proceed to the Installation guide next.
