# Input Component

The Input component is a flexible text input field with built-in validation and multiple states.

## Installation

```bash
npm install @page-kit/components
```

## Import

```tsx
import { Input } from "@page-kit/components";
```

## Basic Usage

```tsx
function MyForm() {
  return <Input placeholder="Enter your name" type="text" />;
}
```

## API Reference

### Props

| Prop          | Type      | Default     | Description                                  |
| ------------- | --------- | ----------- | -------------------------------------------- |
| `type`        | `string`  | `"text"`    | The input type (text, email, password, etc.) |
| `placeholder` | `string`  | `undefined` | Placeholder text                             |
| `disabled`    | `boolean` | `false`     | Whether the input is disabled                |
| `className`   | `string`  | `undefined` | Additional CSS classes                       |

### Events

| Event      | Type                           | Description                        |
| ---------- | ------------------------------ | ---------------------------------- |
| `onChange` | `(event: ChangeEvent) => void` | Fired when the input value changes |
| `onFocus`  | `(event: FocusEvent) => void`  | Fired when the input gains focus   |
| `onBlur`   | `(event: FocusEvent) => void`  | Fired when the input loses focus   |

## Accessibility

The Input component follows WAI-ARIA guidelines:

- Supports keyboard navigation
- Proper focus management
- Screen reader compatible
- Supports aria-labels and descriptions
